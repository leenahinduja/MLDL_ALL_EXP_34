 🚀 Machine Learning & Deep Learning Experiments

This repository contains a collection of hands-on Machine Learning and Deep Learning experiments implemented using Python, Scikit-learn, TensorFlow, and Keras.

The experiments cover fundamental to advanced algorithms including regression, classification, clustering, and neural networks.

---

📌 📊 Experiments List

🔹 1. Linear & Logistic Regression
- Implemented regression techniques on real-world datasets
- Covered both continuous and classification problems

🔹 2. Multiple, Lasso & Ridge Regression
- Explored regularization techniques
- Compared model performance and overfitting control

🔹 3. Decision Tree & Random Forest
- Classification using tree-based models
- Performance comparison and feature importance

🔹 4. K-Nearest Neighbors (KNN)
- Distance-based classification
- Evaluated performance for different K values

🔹 5. Support Vector Machine (SVM)
- Implemented SVM for classification
- Performed hyperparameter tuning

🔹 6. K-Means & Hierarchical Clustering
- Unsupervised learning techniques
- Customer segmentation and cluster visualization

🔹 7. Artificial Neural Network (ANN)
- Built ANN using TensorFlow/Keras
- Solved classification problems

🔹 8. Convolutional Neural Network (CNN)
- Image classification using MNIST dataset
- Learned spatial features

🔹 9. Recurrent Neural Network (RNN) / LSTM
- Applied LSTM on text data (IMDB dataset)
- Sequence learning and sentiment analysis

🔹 10. Autoencoders
- Implemented denoising autoencoder
- Image reconstruction and noise removal
